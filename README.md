# Pick Chunks

## Installation
```
npm install --save-dev @rishabh3112/pc

# OR

yarn add -D @rishabh3112/pc
```

## Usage
Add following to scripts
```js
npx pc --root ./src
npx pc

OR

yarn pc --root ./src
yarn pc
```

## Scripts 
### Serve
serve server at `test/src`
```bash
npm run serve
```

### Build
build project
```bash
npm run build
```

### Test
test project
```bash
npm run test
```

### Development
watch code changes and compile code to lib
```bash
npm run watch
```
